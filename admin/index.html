Painel de anuncios

Atualizar PCs do site

Esta pagina e a entrada segura para editar os anuncios sem alterar manualmente o HTML do site publico.

  1. Abra o Pages CMS e entre com sua conta do GitHub.
  2. Escolha o repositorio mobilytechbr.
  3. Edite a colecao Anuncios de PCs, envie a foto principal, fotos extras, digite os precos apenas com numeros e salve.
  4. Use Adicionais globais para SSDs, armazenamento ou kits de mouse e teclado que podem aparecer em todos os PCs.
  5. Em adicionais globais ou especificos, envie uma foto quando quiser mostrar uma miniatura clicavel ao cliente.
  6. Use Opcionais especificos deste PC somente quando um opcional servir apenas para aquele anuncio.
  7. Se o campo cutout ficar vazio, o GitHub tenta criar o PNG sem fundo automaticamente.
Abrir Pages CMS Voltar para o site
